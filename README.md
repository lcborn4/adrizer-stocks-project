 ```

## Usage
```
git clone https://github.com/lcborn4/adrizer-stocks-project.git
cd adrizer-stocks-project
```
## Loopback Proxy API
PORT 3000
```
loopback backend api
npm install
npm start
```
## React Frontend
### Folder: stocks-frontend-react
PORT 8000

To Run: 
```
cd stocks-frontend-react
